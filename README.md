# 📊 **학생 체형 불균형 분석 대시보드**
데이터 기반의 학생 성장기 체형 변화 추적 및 시각화 프로젝트

## 📌 **1. 프로젝트 개요**
성장기 학생들의 1, 2학기 체형 검사 데이터를 비교하여 신체 균형의 변화를 시각화하고, 특정 부위 간 불균형의 상관관계를 분석하는 웹 기반 대시보드입니다.

## 🎯 **주요 목적**
직관적 이해: 복잡한 수치 데이터를 레이더 차트와 히트맵으로 시각화

정량적 평가: 학기별 개선율 산출을 통한 자세 교정 프로그램 효과 확인

데이터 보안: 개인정보 비식별화 처리를 통한 안전한 포트폴리오 구성

## 🛠 기술 스택 (Tech Stack)
| 분류 | 기술 스택 | 상세 설명 |
| :--- | :--- | :--- |
| **Language** | Python 3.9 | 데이터 전처리 및 분석 메인 언어 |
| **Frontend** | Streamlit | 대화형 웹 대시보드 UI 구현 |
| **Analysis** | Pandas, Plotly | 데이터 핸들링 및 인터랙틱 그래프 시각화 |
| **Deployment** | Streamlit Cloud | GitHub 연동 실시간 웹 서비스 배포 |

## 🚀 **핵심 기능 (Key Features)**
✅ 학년/학교급별 동적 필터링
초/중/고등학교 및 상세 학년별 데이터를 실시간으로 필터링하여 맞춤형 분석 결과 제공

✅ 다각도 시각화 모듈
전신 균형 레이더 차트: 6대 관절(목, 어깨, 허리 등)의 밸런스를 한눈에 파악

학기별 비교 바 차트: 1학기 대비 개선된 부위를 자동으로 계산하여 강조

위험군 분포 파이 차트: 2% 미만 소수 유형을 '기타'로 통합하여 가독성 증대

상관관계 히트맵: 신체 부위 간 불균형 전이 현상을 통계적으로 분석

## 🛡 **데이터 보안 및 전처리**
본 프로젝트는 개인정보 보호를 최우선으로 설계되었습니다.

비식별화: 이름, 생년월일, 학번 등 민감 정보 완전 제거

일반화: 구체적인 학교명을 '초등학교', '중학교' 등으로 치환하여 식별 방지
