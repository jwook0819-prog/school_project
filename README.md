학생 체형 불균형 분석 및 시각화 대시보드
Student Body Posture Disparity Analysis Dashboard

성장기 학생들의 신체 균형 데이터를 분석하여 학기별 변화 양상을 시각적으로 추적하고, 특정 부위 간의 상관관계를 분석하기 위한 데이터 인사이트 도구입니다.

프로젝트 배경 및 목적
데이터 기반 체형 관리: 단순 수치로 나열된 체형 검사 결과를 시각화하여 학생 및 보호자가 신체 변화를 직관적으로 이해하도록 돕습니다.

추세 분석: 1학기 대비 2학기 개선율을 분석하여 체육 활동이나 자세 교정 프로그램의 효과를 정량적으로 평가합니다.

위험군 조기 발견: 전신 균형도와 부위별 연관성을 분석하여 집중 관리가 필요한 위험군을 선별합니다.

기술적 특징 (Technical Highlights)
1. 효율적인 데이터 파이프라인
Data Cleaning: re 라이브러리를 활용한 정규표현식 기반 학년 데이터 표준화 및 결측치 전처리.

Performance: Streamlit @cache_data 데코레이터를 사용하여 대용량 CSV 로딩 속도 최적화 및 서버 부하 감소.

2. 다각도 시각화 설계
Radar Chart: 6개 핵심 부위(목, 어깨, 허리, 엉덩이, 무릎, 발목)의 균형 점수를 방사형으로 시각화하여 전신 밸런스 상태를 직관적으로 표현.

Heatmap Analysis: 부위별 상관계수를 산출하여 특정 부위(예: 허리)의 불균형이 다른 부위(예: 목, 어깨)와 어떻게 연관되는지 통계적으로 제시.

Dynamic Filtering: 학교급(초/중/고) 및 상세 학년별 인터랙티브 필터링 기능을 통해 사용자 맞춤형 분석 지원.

주요 분석 결과 및 인사이트
개선 지표: 1학기 대비 2학기에 가장 크게 개선된 부위를 자동 산출하여 대시보드 상단에 배치 (Insight Card).

기타 그룹화: 분포도가 낮은(2% 미만) 미세 유형들을 '기타'로 통합하여 데이터 노이즈를 제거하고 주요 위험군에 집중할 수 있도록 구성.

불균형 상관관계: 히트맵 분석 결과, 특정 학년군에서는 허리와 골반의 불균형 상관계수가 높게 나타나는 등의 유의미한 패턴 발견.

저장소 구조 및 파일 설명
app.py: Streamlit 기반 웹 애플리케이션 메인 로직 및 레이아웃 설정.

data1.csv / data2.csv: 학기별 정제 데이터셋 (개인정보 비식별화 완료).

requirements.txt: 라이브러리 의존성 정의 (Streamlit, Pandas, Plotly).

Procfile: 웹 서비스 실행을 위한 프로세스 구성 파일 (배포용).

설치 및 실행 방법
